задание по BOM:
Показать схематически из чего состоит BOM, сделать для каждого из API ссылку на отдельную страничку, где будет выводиться значения полей указанных ниже, либо кнопки с действиями.
Для каждой странички вывести вверху кнопки с навигацией, если возможна навигация назад по истории - стрелка назад, тоже самое со стрелкой вперед, также указывать каждый раз длину истории посещений.

Странички
	1. Navigator
	В зависимости от текущего браузера вывести разную информацию:
		a. Chrome
			i. userAgent
			ii. appVersion
		b. Firefox
			i. appName
			ii. cookieEnabled
		c. IE
			i. platform
			ii. geolocation coordinates
	2. Screen
	У каждого пункта - название, краткое объяснение своими словами и значение свойства
		a. availHeight
		b. availWidth
		c. colorDepth
		d. height
		e. pixelDepth
		f. Width
	3. Location
	У каждого пункта название, значение и объяснение своими словами через всплывающее сообщение
		a. hash
		b. search
		c. hostname
		d. href
		e. Pathname
	4. Window
		a. Кнопка для открытия нового окна с указанными размерами через инпут
		b. Вывести в левом углу текущие размеры окна со скроллбаром и без (опционально кроссбраузерно)
		c. (опционально)Перед тем как закрыть открытое через кнопку окно - вызвать диалог с вопросом - действительно ли пользователь хочет уйти со страницы, и если хочет, то закрыть, а если нет, то остаться.
